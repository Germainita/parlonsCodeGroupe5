# Premiers pas avec Django

En cours d'écriture

Premiers pas avec Django vous montre comment développer et déployer des applications Web professionnelles avec Django. Django est le framework Web Python le plus populaire alimentant de grands sites Web tels que Instagram, Pinterest, YouTube, Eventbrite, Bitbucket et Disqus.

Apprenez à concevoir étape par étape trois applications Web progressivement complexes: d'une application Hello World à une application Web de gestion d'évènements robuste avec des comptes d'utilisateurs, des formulaires, des tests et bien plus encore.

Les sujets couverts comprennent:

  • Django 3.0 et Python 3.8
  • Pipenv pour les environnements virtuels
  • L'architecture MVC
  • Les Templates/URLs
  • Les vues basées sur les classes (Class-based views)
  • L'authentification et l'autorisation
  • Le déploiement
  • Le développement guidé par des tests (Test Driven Developmen

# JavaFX par la pratique

En cours d'écriture

Écrire des programmes Java qui vont s’exécuter en console, c’est bien. Mais très souvent en tant que développeur (et surtout pour les utilisateurs finaux), on en veut un peu plus !

Si vous lisez ce livre, c’est que probablement: vous souhaitez découvrir ce que sont que les interfaces graphiques; vous en avez un peu marre de la console :); vous avez bientôt un devoir sur les interfaces graphiques; votre boss/client souhaite avoir des interfaces graphiques au niveau de l’un des projets sur lequel vous travaillez en ce moment; vous êtes tout simplement curieux.

Le besoin est donc là ! Surtout pour ceux qui auront bientôt un devoir :).

Dans ce livre, je vous propose de découvrir comment créer des applications Java disposant d’interfaces graphiques fonctionnelles et professionnelles en utilisant JavaFX.

Les interfaces graphiques (on parle aussi d’IHM pour Interfaces Homme Machine ou de GUI pour Graphical User Interfaces) nous donne la possibilité de permettre aux utilisateurs de nos applications de pouvoir interagir avec ces dernières au travers de boutons, de champs de texte, de liens, d’images et bien d’autres composants graphiques...

# Python par la pratique!

Très souvent lorsque nous nous lançons dans l'apprentissage d'un nouveau langage de programmation, nous commençons par une phase théorique dans laquelle l'on essaie un tant soit peu de comprendre le principe de fonctionnement dudit langage et tout le jargon informatique associé. Ce n'est qu’après cela, que nous pensons généralement à réellement pratiquer.

Mais vu d'un autre angle et étant donné que l'objectif final consiste habituellement à réaliser plus tard quelques projets avec ce fameux langage, je me suis dit qu'il pourrait être également intéressant, de faire pourquoi pas, d'une pierre deux coups. C'est-à-dire apprendre, mais cette fois-ci en pratiquant!

Ainsi, grâce à ce livre, nous apprendrons à programmer en Python en nous amusant à créer une série d'applications et de jeux tout en utilisant des techniques différentes

Nous commencerons en douceur en abordant dans les premières applications les notions les plus simples afin de ne perdre personne en chemin. Ce sera ensuite le moment de s'intéresser à des notions beaucoup plus avancées comme celles des structures de données, des fichiers, des bases de données, des interfaces graphiques, de la création de sites web, du concept de programmation orientée objet en Python... Alors ready?
